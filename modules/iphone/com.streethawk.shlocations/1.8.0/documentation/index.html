Locations Module

Description

StreetHawk's Location module is used for running location based campaigns based on location of your application's user. Also module can be used for calculating the Work and Home locations of your application's user.

Accessing the Locations Module

To access this module from JavaScript, you would do the following:

var shLocation = require("com.streethawk.Locations");

The shLocation variable is a reference to the Module object.

Reference

https://streethawk.freshdesk.com/solution/articles/12000000941-locations

locations.function

  • Start location reporting

    shLocation.startLocationReporting();

  • Stop location reporting

    shLocation.stopLocationReporting();

  • Request permission dialog

    shLocation.startLocationWithPermissionDialog();

locations.property

  • Set or get enable location by default

    shLocation.defaultLocationService = false; var isEnabledByDefault = shLocation.defaultLocationService;

  • Set or get current enable location

    shLocation.locationEnabled = true; var isEnabled = shLocation.locationEnabled;

  • Set or get only report work home location

    shLocation.reportWorkHomeLocationOnly = true; var isReportOnly = shLocation.reportWorkHomeLocationOnly;

Usage

  • Delay asking for Permission

By default StreetHawk library request location permission when call "streethawkinit". In case you may wish to delay asking the user for location permission, you can add the code be as shown below. 

var shLocation = require('com.streethawk.Locations');
//optional to delay ask for permission
shLocation.defaultLocationService = false;
//do streethawkinit and other things.
function buttonLocationEnableClicked(e) {
    //Enable location
    shLocation.locationEnabled = true;
}
